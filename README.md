## 🚀 **Flagship Projects**

### 📈 **1. Advanced Portfolio Optimization Framework**
> Multi-strategy comparison across MVO, Black-Litterman, and Kelly Criterion
- **Features:** 10-asset portfolio, 5,000+ Monte Carlo trials, benchmarking across multiple metrics
- **Tech:** Python, NumPy, cvxpy, Pandas
- **Results:** Quantified risk-adjusted performance with Sharpe/Sortino ratios

### 🎯 **2. ML-Enhanced Portfolio with Covariance Denoising**
> Ledoit-Wolf shrinkage for 12-asset portfolio optimization
- **Features:** Denoised covariance matrix, risk metrics (CVaR, Calmar, Max Drawdown)
- **Tech:** scikit-learn, statsmodels, Pandas
- **Metrics:** Comprehensive performance evaluation

### 💰 **3. Fama-French 5-Factor Analysis**
> Factor regression analysis on 10-stock US equity portfolio
- **Features:** OLS regression, Monte Carlo backtesting, out-of-sample validation
- **Tech:** statsmodels, SciPy, Pandas
- **Results:** Factor exposure and risk attribution analysis

### 🏆 **4. Telecom Loan Default Prediction (IIT-K Hackathon)**
> ML-driven credit risk assessment
- **Achievement:** 🥇 **1st Place – Algo8 Hackathon**
- **Model:** Random Forest with PCA dimensionality reduction
- **Focus:** Optimized for recall; real-world loan dataset
- **Tech:** scikit-learn, XGBoost, Pandas

### 🤖 **5. AI-Powered Defect Detection System**
> Computer vision for manufacturing quality control
- **Internship Project:** Algo8 AI (Jan-May 2020)
- **Tech:** OpenCV, Deep Learning, Image Processing
- **Impact:** Automated defect detection reducing testing time by ~40%
